Key Calculations

All growth rates use compound annual growth rate (CAGR) formulas with monthly averaging for stability when possible. Doubling time calculated using the natural logarithm formula. All increases computed from first available data point: 2023-04-15.

Growth Rate (CAGR)

CAGR = (End_Value / Start_Value)^(1/years) - 1

Doubling Time

ln(2) / ln(1 + growth_rate)
